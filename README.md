A simple console-based Library Management System developed in Java.
The system allows administrators to manage books and users, issue books to users, and return books.

This project demonstrates basic Object-Oriented Programming (OOP) concepts such as classes, encapsulation, and object interaction.

---
**Features:**
- Add new books to the library
- Register new users
- Issue books to users
- Return issued books
- View all books in the library
- Check book availability
